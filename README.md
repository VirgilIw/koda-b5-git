# PRAKTEK GIT

## INISIALISASI GIT

<!-- terminal pakai bash -->

```bash
$ git init
```

## Perubahan sementara (staging)

Menyimpan perubahan sementara dari working directory ke dalam staging area

```sh
$ git add <...dir_file>
$ git add <...dir_folder>
```

## Perubahan Permanen (commit)

Menyimpan seluruh stagin menjadi permanen

<!-- kalau pesan commit panjang pakai yang commit pertama -->

```sh
$ git commit
$ git commit -m "<pesan>"
```
